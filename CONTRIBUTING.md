# Contributing

Thank you for improving the Claude Code Architecture materials.

## Contribution Guidelines

Please keep contributions aligned to the core message:

```text
Claude Code is not your architect.
Claude Code is your architecture amplifier.
```

## Good Contributions

```text
clearer assignment instructions
better source simulations
more realistic production scenarios
improved templates
stronger governance examples
better facilitator notes
```

## Avoid

```text
tool-specific hype
unsafe automation examples
examples that bypass review
examples that remove tests silently
examples that log PII
```

## Pull Request Expectations

```text
[ ] Change is small and focused.
[ ] Documentation is updated.
[ ] Assignment/source files remain consistent.
[ ] No secrets or sensitive data are included.
[ ] The change supports governed engineering.
```
