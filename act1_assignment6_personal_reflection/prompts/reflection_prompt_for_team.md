# Reflection Prompt for Team Workshop

Use this prompt to run the reflection with your own team.

```text
We are preparing to use Claude Code more intentionally in our repositories.

Please answer:

1. Where are we already doing vibe coding?
2. Which tasks are safe for Claude Code today?
3. Which tasks require architect approval?
4. Which repeated review comment should become written guidance?
5. What is one rule we should add to CLAUDE.md?

Then choose one rule that currently lives in people's heads and decide where it should live:
- CLAUDE.md
- tests
- CI/CD
- CODEOWNERS
- PR template
- ADR
```
