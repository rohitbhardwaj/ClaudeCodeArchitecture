# Reflection Quality Rubric

Use this if you want to evaluate participant responses.

## 5 — Excellent

Participant identifies:

```text
a real vibe-coding risk
safe Claude Code tasks
clear approval-required tasks
a repeated review comment
a specific CLAUDE.md rule
an enforcement location beyond CLAUDE.md
```

## 4 — Strong

Participant provides specific examples and one actionable rule.

May not fully define enforcement.

## 3 — Acceptable

Participant gives reasonable answers but rule is somewhat broad.

## 2 — Weak

Participant answers in generic terms without connection to real team practices.

## 1 — Poor

Participant does not distinguish safe tasks from high-risk tasks.
