# Claude Code Adoption Metrics Policy

## Purpose

Ensure Claude Code adoption is measured by engineering outcomes, not usage volume.

## Required Metrics

```text
delivery speed
PR quality
review effort
test quality
defect rate
security findings
rollback rate
developer experience
```

## Prohibited Primary Success Metrics

Do not use these alone as success metrics:

```text
number of prompts
lines of code generated
number of Claude-assisted PRs
developer login count
```

## Key Rule

```text
Measure Claude Code by outcomes, not usage volume.
```
