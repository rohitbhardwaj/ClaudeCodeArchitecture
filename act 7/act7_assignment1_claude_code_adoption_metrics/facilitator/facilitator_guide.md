# Facilitator Guide — Claude Code Adoption Metrics

## Learning Objective

Participants learn to define metrics that show whether Claude Code improves engineering outcomes.

## Setup Script

Say:

```text
Leadership will ask whether Claude Code is helping.
The wrong answer is: usage is up.
The right answer is: delivery is faster while quality, safety, and reviewability hold.
```

## 5-Minute Live Version

| Segment | Time |
|---|---:|
| Explain outcome vs usage metrics | 1 min |
| Participants fill worksheet | 2 min |
| Show strong sample | 1 min |
| Debrief | 1 min |

## 15-Minute Workshop Version

| Segment | Time |
|---|---:|
| Explain metrics categories | 2 min |
| Fill worksheet | 5 min |
| Review sample data | 4 min |
| Draft leadership scorecard | 3 min |
| Debrief | 1 min |

## Facilitation Prompts

Ask:

```text
What would prove Claude Code is helping?
What would prove it is creating risk?
Which metric could be gamed?
Which guardrail metric matters most?
What would leadership need to see monthly?
```

## Strong Closing Line

Measure Claude Code by outcomes, not usage volume.
