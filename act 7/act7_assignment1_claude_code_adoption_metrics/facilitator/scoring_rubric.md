# Scoring Rubric — Claude Code Adoption Metrics

## 5 — Excellent

Participant defines balanced success and guardrail metrics with good/bad signals.

Strong answer includes:

```text
cycle time
PR review quality
review effort
meaningful tests
defect rate
security findings
rollback rate
developer experience
```

## 4 — Strong

Participant covers most metrics but misses one guardrail.

## 3 — Acceptable

Participant includes outcome metrics but weak good/bad signal definitions.

## 2 — Weak

Participant focuses mostly on speed and adoption volume.

## 1 — Poor

Participant measures only usage.
