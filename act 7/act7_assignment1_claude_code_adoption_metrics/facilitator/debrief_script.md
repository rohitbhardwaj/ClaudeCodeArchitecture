# Debrief Script

## Opening

This exercise is about measuring value, not novelty.

## Key Message

Do not optimize for:

```text
more prompts
more generated code
more AI-assisted PRs
```

Optimize for:

```text
faster delivery with stable or better quality
smaller clearer PRs
meaningful tests
lower defects
fewer security findings
low rollback rate
better developer experience
```

## Closing Line

Measure Claude Code by outcomes, not usage volume.
