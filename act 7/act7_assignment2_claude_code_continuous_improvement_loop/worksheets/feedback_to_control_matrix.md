# Feedback-to-Control Matrix

| Feedback pattern | Control to update |
|---|---|
| Large Claude-generated PRs | Small PR rules in CLAUDE.md and PR template |
| Missing failure-path tests | Test expectations in CLAUDE.md and CI |
| PII logging attempts | Security policy and PII scan |
| API error-code changes | OpenAPI contract tests and API approval rule |
| Missing rollback plan | PR template and rollback runbook |
| Weak prompt context | Prompt templates and training |
| Repeated architecture comments | Architecture boundaries and review checklist |
| Confusing PR descriptions | PR template and Claude disclosure rule |
| Dependency issues | Dependency policy and scan |
| Reviewers unsure what to check | Reviewer checklist and kata |
