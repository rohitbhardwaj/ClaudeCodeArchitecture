# CLAUDE.md

## Claude Code Rules

- Analyze before editing.
- Keep PRs small.
- Add failure-path tests for behavior changes.
- Include rollback plan for production-impacting changes.
- Do not log PII.
- Preserve public API behavior unless approved.

## Monthly Improvement Note

Rules are reviewed monthly based on Claude-assisted PRs, review comments, CI failures, security findings, incidents, and feedback.
