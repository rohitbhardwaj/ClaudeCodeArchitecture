# CLAUDE.md — Continuous Improvement Rule

```markdown
## Continuous Improvement

This repository updates Claude Code guidance monthly based on:
- Claude-assisted PR review
- review comment themes
- failed CI checks
- security findings
- rollbacks/incidents
- developer feedback
- reviewer feedback
- repeated prompt failures

Repeated failures must become:
- stronger CLAUDE.md rules
- reusable prompt templates
- PR checklist updates
- stronger CI gates
- security policy updates
- new team katas
```
