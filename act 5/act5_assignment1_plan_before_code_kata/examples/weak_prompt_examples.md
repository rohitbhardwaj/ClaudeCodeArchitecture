# Weak Prompt Examples

## Weak Prompt 1

```text
Fix the checkout bug.
```

Why weak:

```text
Too broad. Allows immediate implementation.
```

## Weak Prompt 2

```text
Fix the checkout bug and add tests.
```

Why weak:

```text
Still starts with implementation.
```

## Weak Prompt 3

```text
Update CheckoutService to fix the bug.
```

Why weak:

```text
Assumes the root cause and target file before analysis.
```
