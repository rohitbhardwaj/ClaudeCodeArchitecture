# Assignment Instructions

## Scenario

Claude is asked:

```text
Add promotional discount codes during checkout.
```

## Architect Task

Identify what this change could impact before coding.

## Impact Areas

```text
User-visible behavior
API contract
Pricing
Tax
Payment
Refunds
Fraud/abuse
Data model
Analytics
Support
Observability
Tests
Rollback
Required approvals
```

## Key Lesson

```text
Before Claude changes code, architects must understand blast radius.
```
