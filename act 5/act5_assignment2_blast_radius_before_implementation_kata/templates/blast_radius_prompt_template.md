# Blast Radius Prompt Template

```text
Before implementing [feature/change], perform a blast-radius assessment.

Do not edit files yet.

Assess impact on:
- user-visible behavior
- API contract
- pricing
- tax
- payment
- refunds
- fraud/abuse
- data model
- analytics
- support
- observability
- tests
- rollback
- required approvals

Return:
1. impact area
2. risk or consideration
3. files likely involved
4. tests needed
5. approval owners
6. smallest safe implementation sequence

Wait for approval before coding.
```
