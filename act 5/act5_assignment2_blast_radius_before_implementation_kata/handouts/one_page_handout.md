# One-Page Handout — Blast Radius Before Implementation Kata

## Scenario

```text
Add promotional discount codes during checkout.
```

## Key Lesson

```text
Before Claude changes code, architects must understand blast radius.
```

```text
User-visible behavior
API contract
Pricing

```

## Strong Rule

```text
No blast-radius analysis, no implementation.
```
