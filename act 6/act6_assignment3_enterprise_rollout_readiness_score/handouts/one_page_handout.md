# One-Page Handout — Enterprise Rollout Readiness Score

## Key Lesson

```text
Enterprise rollout should be based on readiness evidence, not enthusiasm.
```

## Score Interpretation

| Total score | Readiness |
|---:|---|
| 40–50 | Ready for governed rollout |
| 30–39 | Pilot with restrictions |
| 20–29 | Limited use only |
| Below 20 | Not ready for production repositories |

## Readiness Areas

```text
CLAUDE.md
CODEOWNERS
PR template
CI tests
API contract tests
Security/PII scanning
Dependency scanning
Rollback
Developer training
Reviewer training
```
