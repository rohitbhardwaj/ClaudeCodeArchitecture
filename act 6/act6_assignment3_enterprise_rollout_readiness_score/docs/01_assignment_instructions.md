# Assignment Instructions

## Scenario

A VP of Engineering asks:

```text
Are we ready to allow Claude Code on production repositories?
```

## Architect Task

Score the organization’s readiness from 1 to 5 in each area.

Then provide a rollout recommendation.

## Score Interpretation

| Total score | Readiness |
|---:|---|
| 40–50 | Ready for governed rollout |
| 30–39 | Pilot with restrictions |
| 20–29 | Limited use only |
| Below 20 | Not ready for production repositories |

## Key Lesson

```text
Enterprise rollout should be based on readiness evidence, not enthusiasm.
```
