# Facilitator Guide — Enterprise Rollout Readiness Score

## Learning Objective

Participants learn to make rollout decisions based on evidence rather than enthusiasm.

## Setup Script

Say:

```text
A VP asks whether the organization is ready to allow Claude Code on production repositories.
The wrong answer is based on excitement.
The right answer is based on evidence.
```

## 5-Minute Live Version

| Segment | Time |
|---|---:|
| Explain scorecard | 1 min |
| Participants score quickly | 2 min |
| Show sample recommendation | 1 min |
| Debrief | 1 min |

## 15-Minute Workshop Version

| Segment | Time |
|---|---:|
| Explain readiness areas | 2 min |
| Inspect evidence | 4 min |
| Score worksheet | 4 min |
| Draft VP recommendation | 3 min |
| Debrief | 2 min |

## Facilitation Prompts

Ask:

```text
What evidence proves readiness?
What is missing?
What should be allowed now?
What should be restricted?
What must improve before broad rollout?
```

## Strong Closing Line

Enterprise rollout should be based on readiness evidence, not enthusiasm.
