# Debrief Script

## Opening

This exercise turns governance into an executive readiness decision.

## Key Message

Do not ask:

```text
Are we excited about Claude Code?
```

Ask:

```text
Do we have evidence that production use is governed?
```

## Strong Recommendation Pattern

For a 32/50 score:

```text
Pilot with restrictions.
Allow read-only analysis, documentation, test drafting, and small approved changes.
Do not allow autonomous production-impacting changes yet.
```

## Improvements

```text
update CLAUDE.md
enforce CODEOWNERS
improve PR template
add API/security/dependency scans
train reviewers
```

## Closing Line

Enterprise rollout should be based on readiness evidence, not enthusiasm.
