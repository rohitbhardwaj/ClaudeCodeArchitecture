# Assignment Prompt

```text
A VP of Engineering asks:

"Are we ready to allow Claude Code on production repositories?"

Score readiness from 1 to 5 for:
- CLAUDE.md exists and is current
- CODEOWNERS configured
- PR template includes Claude disclosure
- Tests required in CI
- API contract tests exist
- Security/PII scanning exists
- Dependency scanning exists
- Rollback process is defined
- Developers trained on plan-first workflow
- Reviewers trained on AI-generated diffs

Return:
1. score per area
2. evidence
3. improvement needed
4. total score
5. rollout recommendation
```
