# Assignment Instructions

## Scenario

Your organization is rolling out Claude Code across multiple production repositories.

You need to define where governance controls should live.

## Architect Task

Map each control to the right repository or operating artifact.

## Controls to Map

```text
Repo-specific Claude rules
Architecture boundaries
Approval owners
PR disclosure requirements
Test requirements
API contract rules
Dependency review rules
Security/PII rules
Rollback expectations
CI enforcement
```

## Key Lesson

```text
Governance must live where developers and Claude Code can actually see it.
```
