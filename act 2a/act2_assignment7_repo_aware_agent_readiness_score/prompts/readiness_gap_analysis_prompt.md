# Readiness Gap Analysis Prompt

```text
Analyze readiness gaps for this repository.

Do not modify files.

Focus on:
1. What rules are tribal knowledge?
2. What would Claude likely misunderstand?
3. Which files lack ownership?
4. Which tests are missing?
5. Which API contracts are undocumented?
6. Which security/logging rules are missing?
7. Which CI/CD gates should be added?
8. Which rollback patterns are absent?

Return a prioritized improvement backlog.
```
