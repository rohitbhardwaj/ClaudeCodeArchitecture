# Assignment Instructions

## Scenario

Your team currently treats Claude Code like an advanced autocomplete tool.

Developers use prompts like:

```text
Write this function.
Fix this bug.
Add tests.
Clean this file.
```

But Claude Code can reason across files, understand repository context, generate changes, and assist with engineering workflow tasks.

## Architect Task

Classify each use case into one of three levels:

```text
Level 1 — Code helper
Level 2 — Repo-aware assistant
Level 3 — Engineering workflow agent
```

## Classification Worksheet

Use the worksheet in:

```text
worksheets/participant_classification_worksheet.md
```

## Reflection Question

```text
Where is your team still treating Claude Code like autocomplete when it should be treated like a repo-aware agent?
```

## Output

Each participant should leave with:

```text
1. One Level 1 use case suitable for low-risk usage.
2. One Level 2 use case that requires repo context.
3. One Level 3 use case that requires workflow controls.
4. One team reflection on where governance must improve.
```
