# Act 2 Assignment 1 — From Coding Assistant to Repo-Aware Agent

## Theme

Claude Code is not just autocomplete.

It can reason across files, understand repository context, generate changes, run commands, summarize pull requests, and participate in engineering workflows.

## Core Lesson

```text
The more repo-aware Claude becomes, the more architecture-aware your controls must become.
```

## Scenario

Your team currently treats Claude Code like an advanced autocomplete tool.

Developers use prompts like:

```text
Write this function.
Fix this bug.
Add tests.
Clean this file.
```

But Claude Code can operate at multiple levels:

```text
Level 1 — Code helper
Level 2 — Repo-aware assistant
Level 3 — Engineering workflow agent
```

## Architect Task

Participants classify Claude Code use cases into three levels.

They then reflect on where their team is still treating Claude Code like autocomplete when it should be treated like a repo-aware agent.
