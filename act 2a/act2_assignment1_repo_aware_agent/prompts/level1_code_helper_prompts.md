# Level 1 — Code Helper Prompt Examples

## Utility Function

```text
Write a Java function that validates whether a string is a non-empty SKU.

Constraints:
- Return false for null or blank values.
- Do not use external libraries.
- Include simple unit test examples.
```

## Small Test Case

```text
Generate a unit test for this helper method.

Focus only on:
- null input
- blank input
- valid input

Do not modify production code.
```

## Local Explanation

```text
Explain what this method does line by line.

Do not suggest changes yet.
```

## Governance Note

Level 1 prompts are lower risk, but generated code still requires review.
