# Assignment Instructions

## Scenario

Your team wants to introduce Claude Code into the software delivery lifecycle.

The risk is that teams adopt Claude Code randomly:

```text
one developer uses it for code
another uses it for tests
another uses it to summarize PRs
another uses it during incidents
```

Without a lifecycle map, usage becomes inconsistent and hard to govern.

## Architect Task

Map where Claude Code can help across the lifecycle.

For each delivery stage, identify:

```text
1. How Claude Code can help.
2. What risks this creates.
3. What guardrail is needed.
```

## Delivery Stages

```text
Discovery
Design
Coding
Testing
Review
Documentation
Modernization
Incident response
```

## Reflection Question

```text
Which delivery stage gives your team the highest value with the lowest risk?
```
