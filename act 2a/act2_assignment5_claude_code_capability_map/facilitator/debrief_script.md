# Debrief Script

## Opening

The most obvious use of Claude Code is coding.

But the safest first use may not be coding.

## Key Message

Claude Code can help across the lifecycle:

```text
discovery
design
coding
testing
review
documentation
modernization
incident response
```

But every lifecycle stage has a different risk profile.

## Important Contrast

Discovery may be low risk because Claude is mostly reading and explaining.

Coding may be higher risk because Claude is changing behavior.

Incident response may be high risk because speed and pressure can amplify mistakes.

## Governance Message

Architects should decide:

```text
where Claude helps
what risk appears
what guardrail is required
who remains accountable
```

## Closing Line

Claude Code is most valuable when placed intentionally into the delivery lifecycle, not randomly invoked during coding.
