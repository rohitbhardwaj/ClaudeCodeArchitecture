# Modernization Prompts

## Legacy Module Analysis

```text
Analyze this legacy module for modernization.

Do not modify files.

Return:
1. Responsibilities.
2. Dependencies.
3. Data flow.
4. Risk hotspots.
5. Test gaps.
6. Suggested seams.
7. Characterization tests.
8. Incremental migration plan.
9. First three safe PRs.
```

## Safe Refactor Plan

```text
Create a behavior-preserving refactor plan.

Do not code yet.

Include:
1. Current behavior.
2. Tests that protect it.
3. Missing characterization tests.
4. Small refactor steps.
5. Rollback plan.
```
