# Design Prompts

## Draft ADR

```text
Draft an ADR for the proposed design.

Include:
1. Context.
2. Decision.
3. Options considered.
4. Tradeoffs.
5. Risks.
6. Consequences.
7. Open questions.

Do not present this as approved.
```

## Compare Options

```text
Compare three design options for this problem.

For each option include:
1. Summary.
2. Benefits.
3. Risks.
4. Operational impact.
5. Security impact.
6. Testing impact.
7. Recommendation.

Label assumptions clearly.
```
