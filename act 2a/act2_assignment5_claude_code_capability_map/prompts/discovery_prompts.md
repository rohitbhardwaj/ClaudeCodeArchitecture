# Discovery Prompts

## Explain Module

```text
Analyze this module in discovery mode.

Do not modify files.

Return:
1. Main responsibilities.
2. Entry points.
3. Key dependencies.
4. Data flow.
5. Tests that cover it.
6. Risk hotspots.
7. Open questions for code owners.
```

## Map Dependencies

```text
Map dependencies for the checkout module.

Do not modify files.

Include:
1. Internal dependencies.
2. External dependencies.
3. Persistence dependencies.
4. Test dependencies.
5. Potential hidden coupling.
```
