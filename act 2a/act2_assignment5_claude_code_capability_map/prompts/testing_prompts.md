# Testing Prompts

## Generate Test Plan

```text
Create a test plan for this module.

Do not modify files yet.

Include:
1. Existing tests.
2. Missing happy-path tests.
3. Missing failure-path tests.
4. Edge cases.
5. Security-sensitive cases.
6. Characterization tests needed before refactoring.
```

## Draft Tests

```text
Draft unit tests for the approved behavior.

Constraints:
- Do not change production code.
- Do not weaken existing tests.
- Include failure paths.
- Explain what behavior each test protects.
```
