# Prompts for Human-Owned Decisions

## Architecture Options, Not Final Decision

```text
Prepare architecture options for human review.

Do not choose the final architecture.

For each option include:
1. Summary.
2. Benefits.
3. Tradeoffs.
4. Operational impact.
5. Security impact.
6. Cost impact.
7. Team skill assumptions.
8. Risks.
9. Open questions.
```

## API Contract Change Analysis

```text
Analyze the proposed API contract change for human approval.

Do not modify files.

Include:
1. Current contract.
2. Proposed change.
3. Backward compatibility impact.
4. Client impact.
5. Monitoring/support impact.
6. Migration plan.
7. Tests required.
8. Approval required.
```

## Rollback Decision Support

```text
Prepare rollback decision support.

Do not decide rollback.

Include:
1. Current production symptoms.
2. Customer impact.
3. Risk of rollback.
4. Risk of staying forward.
5. Data migration concerns.
6. Feature flag options.
7. Recommended evidence to review.
8. Human decision owners.
```
