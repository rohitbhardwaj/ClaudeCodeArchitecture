# Prompts for Tasks Claude Can Automate

## Explain Code

```text
Explain how this module works.

Do not modify files.

Include:
1. Main responsibilities.
2. Entry points.
3. Dependencies.
4. Data flow.
5. Tests that cover it.
6. Uncertainties.
```

## Suggest Refactor Plan

```text
Suggest a refactoring plan.

Do not modify files.

Include:
1. Current responsibilities.
2. Code smells.
3. Proposed small steps.
4. Risks.
5. Tests needed.
6. Files likely to change.
```

## Summarize Incident Evidence

```text
Summarize the incident evidence.

Do not decide root cause.

Organize:
1. Timeline.
2. Logs.
3. Recent diffs.
4. Failing tests.
5. Metrics.
6. Hypotheses with evidence.
7. Open questions.
```
