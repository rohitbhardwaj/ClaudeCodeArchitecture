# Act 2 Assignment 6 — Human Judgment vs Claude Automation

## Theme

Claude Code can accelerate engineering work, but humans remain accountable for architectural judgment.

## Scenario

Your organization is deciding what Claude Code should automate and what humans must still decide.

## Architect Task

Participants separate responsibilities into three categories:

```text
Claude can automate
Human must decide
Shared
```

## Responsibilities

```text

Choose architecture style
Generate unit tests
Approve API contract change
Suggest refactor plan
Decide payment behavior
Draft ADR
Merge PR
Identify security risks
Approve production release
Summarize incident evidence
Decide rollback
```

## Reflection Question

```text
What decision are you currently allowing tools to influence too much?
```
