# Act 2 Assignment 3 — Task Boundary Classification

## Theme

Claude Code should not have one permission level.

Different engineering actions require different authority.

## Scenario

A development team wants to use Claude Code for several tasks.

Participants must classify each task into one of three categories:

```text
Safe now
Needs approval
Not allowed
```

## Key Lesson
