# Scoring Rubric — Task Boundary Classification

## 5 — Excellent

Participant correctly classifies nearly all tasks and explains risk using:

```text
blast radius
security
public API
payment behavior
test integrity
data safety
review governance
```

## 4 — Strong

Participant classifies most tasks correctly and explains at least three risk categories.

## 3 — Acceptable

Participant gets obvious safe/not-allowed tasks right but misses context-sensitive tasks.

## 2 — Weak

Participant treats most tasks as safe and misses approval boundaries.

## 1 — Poor

Participant does not distinguish task authority levels.
