# Safe-Now Prompt Examples

## Explain Checkout

```text
Explain how checkout works.

Do not modify files.

Include:
1. Entry point.
2. Main service flow.
3. Pricing path.
4. Payment path.
5. Persistence path.
6. Tests that cover this behavior.
```

## Summarize PR

```text
Draft a PR summary for the current changes.

Do not invent results.

Include:
1. What changed.
2. Files changed.
3. Tests run.
4. Risks.
5. Follow-up work.
```

## Suggest Refactoring Plan

```text
Suggest a refactoring plan for this module.

Do not modify files.

Return:
1. Current responsibilities.
2. Code smells.
3. Small-step plan.
4. Risk level.
5. Tests needed.
```
