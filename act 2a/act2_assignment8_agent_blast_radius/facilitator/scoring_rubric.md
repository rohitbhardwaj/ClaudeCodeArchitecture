# Scoring Rubric — Agent Blast Radius Exercise

## 5 — Excellent

Participant identifies all major blast-radius categories and provides specific impacts.

Includes:

```text
user behavior
API
pricing
tax
payment
refunds
fraud
data model
analytics
support
observability
tests
rollback
approvals
```

## 4 — Strong

Participant identifies most categories but may miss one or two such as analytics, support, or rollback.

## 3 — Acceptable

Participant identifies obvious technical areas but misses financial, support, or operational impact.

## 2 — Weak

Participant focuses mainly on code or pricing.

## 1 — Poor

Participant approves implementation without blast-radius analysis.
