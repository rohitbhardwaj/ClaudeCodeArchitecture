# Debrief Script

## Opening

The request was:

```text
Add support for promotional discount codes during checkout.
```

At first glance, that sounds like a small feature.

But after blast-radius analysis, we can see it touches much more than code.

## Key Message

Discount codes affect:

```text
customer experience
pricing
tax
payment
refunds
fraud
analytics
support
observability
data
testing
rollback
approvals
```

## Architecture Point

A coding agent may implement the first obvious path.

An architect must identify the system impact before implementation begins.

## Governance Message

Claude Code should not start with:

```text
implement promotional discounts
```

It should start with:

```text
analyze the blast radius and wait for approval
```

## Closing Line

Before Claude changes code, architects must understand the blast radius.
