# Blast Radius Review Checklist

## Business Behavior

```text
[ ] User-visible behavior identified
[ ] Product owner approval needed?
[ ] Support impact identified?
[ ] Analytics impact identified?
```

## Financial Behavior

```text
[ ] Pricing impact identified
[ ] Tax impact identified
[ ] Payment amount impact identified
[ ] Refund impact identified
[ ] Finance approval needed?
```

## Technical Contract

```text
[ ] API response fields impacted?
[ ] Error/rejection reason codes impacted?
[ ] Data model changes needed?
[ ] Backward compatibility considered?
```

## Security and Abuse

```text
[ ] Coupon brute force considered?
[ ] Eligibility rules considered?
[ ] Rate limiting needed?
[ ] PII/logging risk considered?
[ ] Security approval needed?
```

## Delivery Safety

```text
[ ] Tests identified?
[ ] Feature flag required?
[ ] Rollback plan defined?
[ ] Monitoring and alerts defined?
[ ] Required approvals listed?
```
