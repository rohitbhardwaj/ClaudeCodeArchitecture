# Act 2 Assignment 8 — The Agent Blast Radius Exercise

## Theme

Before Claude changes code, architects must understand the blast radius.

## Scenario

Claude Code is asked to make a change:

```text
Add support for promotional discount codes during checkout.
```

Before allowing implementation, participants estimate the blast radius.

## Architect Task

Identify all affected areas:

```text
1. User-visible behavior
2. API contract
3. Pricing
4. Tax
5. Payment
6. Refunds
7. Fraud/abuse
8. Data model
9. Analytics
10. Support
11. Observability
12. Tests
13. Rollback
14. Required approvals
```

## Key Lesson
